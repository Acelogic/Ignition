Ignition

A modern, native macOS launch agent manager. Browse, control, create, and monitor launchd services — all from one app.

A complete LaunchControl replacement — free and open source.

📋

Agent Management

Browse all five launchd domains. Start, stop, load, and unload agents with one click. Search and filter by label or program path.

Core
📄

Live Log Viewer

Real-time log tailing using kqueue file monitoring. Auto-scroll, search, copy. Stderr lines highlighted in red for quick debugging.

Debugging
✏️

In-App Plist Editor

Recursive key-value editor with type-aware controls. Auto-backup, admin password prompt for global domains, read-only for system plists.

Editing
🧙

Create Agent Wizard

Five templates: Run Script, Watch Folder, Run at Login, Scheduled Task, and Blank. Three-step flow with live XML preview.

Creation
🔥

Menu Bar Quick Access

Pin agents to the menu bar for always-accessible status and control. Toggle load/unload without opening the main window.

Convenience
💓

Health Dashboard

Live CPU and memory monitoring. Crash loop detection. Problem agent alerts. Overview cards with at-a-glance system health.

Monitoring
📜

Activity History

SQLite-backed event log. Tracks start, stop, crash, load, and unload events. Per-agent and global timelines with relative timestamps.

Tracking
🔔

Notifications

macOS notifications when pinned agents crash or stop. Configurable alerts using the native UserNotifications framework.

Alerts
📦

Import / Export

Export agents as plist bundles. Import from files or folders. One-click backup of all user agents to timestamped directories.

Quick Start

# Install XcodeGen
brew install xcodegen

# Clone the repository
git clone https://github.com/Acelogic/Ignition.git
cd Ignition

# Generate project and build
xcodegen generate
xcodebuild -scheme Ignition build

# Or open in Xcode
open Ignition.xcodeproj
